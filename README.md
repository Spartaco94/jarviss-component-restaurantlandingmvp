# RestaurantLandingMVP - Verde Radice

## Descrizione
Componente landing page modulare per ristoranti vegani con sistema di prenotazione e showcase menù.

## Struttura
- `index.html`: Struttura principale della pagina
- `styles.css`: Fogli di stile responsive
- `menu.js`: Gestione dinamica del menù
- `booking.js`: Logica di prenotazione

## Requisiti
- Supporto browser moderni
- JavaScript abilitato

## Personalizzazione
1. Modificare i dati in `menu.js`
2. Adattare colori in `styles.css`
3. Configurare endpoint API in `booking.js`

## Implementazione
```bash
# Clonare la repository
git clone https://github.com/verderadice/landing-mvp

# Aprire index.html in un browser
```
